## ğŸ” Troubleshooting

### Profile does not appear after download
- Make sure you are using **Safari**
- Try downloading the file again from the official release page

### Cannot install the profile
- Ensure your iOS version is up to date

### Installation failed or no response
- Restart your device and try again
- Re-download the configuration profile from the official source
