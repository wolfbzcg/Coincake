# 🤖 Coincake Android 安装指南

本指南将帮助您在 Android 设备上 **安全、正确** 地安装 Coincake 官方应用。

Coincake Android 版本 **仅通过官方 GitHub Releases 渠道发布**，请勿通过第三方来源下载安装。

---

## ✅ 安装前准备

在开始安装前，请确认以下条件：

- Android 系统版本 **Android 8.0 或更高**
- 设备已连接稳定的网络
- 设备有足够的存储空间

---

## 📥 第一步：下载 APK 安装包

1. 打开 Coincake 官方下载页面：  
   👉 https://dao-coincake.github.io/coincake-platform/download/zh-CN.html
2. 点击 **“下载最新 APK”**
3. 等待 APK 文件下载完成

> ⚠️ 请务必确认 APK 来自 Coincake 官方下载页。

---

## ⚙️ 第二步：允许安装来自此来源的应用

根据您的 Android 系统版本，可能会出现安全提示。

### 常见操作流程：
1. 点击刚刚下载完成的 APK 文件  
2. 系统提示“为安全起见，您的手机目前无法安装此应用”  
3. 点击 **设置**  
4. 启用 **允许来自此来源的应用**  
5. 返回安装界面

> 安装完成后，您可以随时在系统设置中关闭此权限。

---

## 📲 第三步：安装应用

1. 点击 **安装**  
2. 等待系统完成安装过程  
3. 安装完成后，点击 **打开**

Coincake 应用图标将出现在您的主屏幕或应用列表中。

---

## 🔄 第四步：保持应用为最新版本

为确保最佳使用体验与安全性：

- 建议定期访问官方发布页更新应用  
- 较旧版本可能会逐步停止支持  
- 所有更新均通过 GitHub Releases 发布

---

## 🔎 常见问题排查

### ❓ 无法安装或安装被阻止
- 请确认已启用“允许来自此来源的应用”
- 确保 APK 文件已完整下载
- 尝试重启设备后重新安装

### ❓ 应用安装成功但无法打开
- 请确认系统版本满足最低要求
- 尝试重新安装最新版本

---

## 🔐 安全提醒

- 请仅从 Coincake 官方页面下载 APK  
- Coincake **不会** 索取私钥、助记词或任何敏感信息  
- 请勿安装任何经过修改的第三方 APK 文件  

如需了解更多安全信息，请参阅：
