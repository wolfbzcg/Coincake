# Security Policy

Coincake takes security issues seriously and is committed to protecting users and platform integrity.

---

## Response Process

- We aim to acknowledge security reports within **48 hours**
- Confirmed issues will be addressed in subsequent updates or releases
- Security-related fixes may be released without prior public disclosure

---

## Recommendations for Users

- Only download Coincake from the official GitHub Releases page
- Do not share private keys or sensitive credentials
- Be cautious of impersonation or unofficial distribution channels
